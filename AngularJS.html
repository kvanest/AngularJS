It is currently maintained by Google and a community of developers. The latest update was on September 29, 2015.
How reputable are the developers?
Very much. One of the developers, Misko Hevery, works at Google.
How long has the project survived?
AngularJS was initially released in 2009, developed by Brat Tech LLC.
What are the competitors?
Backbone.js and Ember.js are the two main competitors.
Why is this project better than competitors?
Angular has the largest number of YouTube Results, Github stars and contributors, and Chrome extension users. Also, according to Google Trends, Angular has grown exponentially in intereste level, while Backbone and Ember have remained steady. Angular also uses two-way binding, which means you never have to write the code yourself. Angular is also backed by Google, meaning it is always growing and has developer productivity.
How big is the learning curve?
Easy to moderate. Becuase of the functionality, it is easy to use components, define new elements, attributes, and behaviors.
How good is the documentation?
Documentation is good. It gives you examples and short paragraphs that discuss whatever it is you want to work on. I don't think it's as extensive as other frameworks out there.
How compatible is it
Fairly compatible. Angular works in Chrome and Firefox, but does not work in Internet Explorer versions under 11.
